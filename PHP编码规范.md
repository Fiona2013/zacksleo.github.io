## PSR
>PSR 是 PHP Standard Recommendations 的简写，由 PHP FIG 组织制定的 PHP 规范，是 PHP 开发的实践标准。

**PHP编码遵循PSR的建议**

主要包括以下几个方面:

### PSR-1 基础编码规范
### PSR-2 编码风格规范
### PSR-3 志接口规范
### PSR-4 自动加载规范
### PSR-6 缓存接口规范
### PSR-7 HTTP 消息接口规范

详细内容见: https://psr.phphub.org/ 和 https://github.com/PizzaLiu/PHP-FIG

## 注意
代码审核时将遵守以上规范
